## Developement

### Installation
First clone this repo, then simply run `$ yarn install` or `$ npm i`

### Developement
1. Run developement server `$ yarn start` 
2. Build
    - run for Android `$ yarn install:dev` or `$ react-native run-android`
    - run for iOS  `$ react-native run-ios`
